예시문항 1 난이도: 4/10

\(1 < n < 10\)인 자연수 \(n\)에 대하여 \(n^{2} - 5n - 6\)의 \(n\)제곱근 중에 음의 실수가 존재하도록 하는 모든 \(n\)의 값의 합을 구하시오.

암기하자! 아래의 풀이를 꼭 기억하고 유연하게 서술할 수 있도록 연습할 것

다음과 같이 경우를 나눈다.

\(i\). \(n\)이 홀수이고 (주어진 식) \(< 0\)일 때

(주어진 식) \(= n^{2} - 5n - 6\)이므로
\(n^{2} - 5n - 6 < 0\)
\((n - 6)(n + 1) < 0\)
\(-1 < n < 6\)
문제에서 주어진 조건 \(1 < n < 10\)까지 반드시 고려한다.
\(\therefore 1 < n < 6\)
이를 만족하는 "홀수" \(n\)을 구해야 함을 잊지 말자.
따라서 자연수 \(n\)의 값은 \(3\), \(5\)

\(ii\). \(n\)이 짝수이고 (주어진 식) \(> 0\)일 때

\(n^{2} - 5n - 6 > 0\)
\((n - 6)(n + 1) > 0\)
\(n < -1\) 또는 \(n > 6\)
문제에서 주어진 조건 \(1 < n < 10\)까지 반드시 고려한다.
\(\therefore 6 < n < 10\)
이를 만족하는 "짝수" \(n\)을 구해야 함을 잊지 말자.
따라서 자연수 \(n\)의 값은 \(8\)

\(i\), \(ii\)에 의하여 가능한 자연수 \(n\)의 값은 \(3\), \(5\), \(8\)이고 그 합은 \(16\)
